# 📧 Настройка Resend и погоды (Open-Meteo)

- **Resend** — отправка email (3000 писем/мес бесплатно)
- **Open-Meteo** — прогноз погоды (бесплатно, без API ключа)

---

## 📧 Часть 1: Настройка Resend для отправки email

### 1. Зарегистрируйтесь на Resend

https://resend.com

- Нажмите **Sign Up** → через GitHub
- Подтвердите email

### 2. Получите API Key

1. Зайдите в **Settings** → **API Keys**
2. **Create API Key**
3. Название: `Smart Home App`
4. Скопируйте ключ (начинается с `re_...`)

### 3. Добавьте переменную на Render

1. Зайдите на: https://dashboard.render.com
2. Ваш сервис `smart-home-api` → **Environment**
3. **Add Environment Variable**:

| Имя | Значение |
|-----|----------|
| `RESEND_API_KEY` | `re_xxxxxxxxxxxxxxxxxxxxx` (ваш ключ) |

4. **Save Changes**

Сервис автоматически перезапустится.

## 🌤️ Часть 2: Настройка погоды (Open-Meteo)

Погода использует **Open-Meteo API** — бесплатный сервис, не требующий регистрации!

### Преимущества Open-Meteo:
- ✅ **Бесплатно** — без ограничений
- ✅ **Без API ключа** — не нужно регистрироваться
- ✅ **Работает в России** — в отличие от OpenWeatherMap
- ✅ **Точные данные** — используется метеомодель ECMWF

### Как работает:
1. При запросе города происходит геокодинг (поиск координат)
2. По координатам получаем текущую погоду и прогноз на 3 дня
3. Данные обновляются автоматически

**Никаких дополнительных настроек не требуется!** Просто укажите город в интерфейсе.

## 🔍 Если письма не приходят

### Проверьте спам
Письма могут попасть в папку "Спам"

### Проверьте email
Убедитесь, что ввели правильный email при регистрации

### Проверьте логи
Если видите ошибку:
```
❌ Ошибка отправки email: ...
```

Проверьте:
1. Правильность `RESEND_API_KEY`
2. Нет ли опечаток в переменной на Render
